## Getting Started in 3 Minutes

### Step 1: Setup API Key (1 minute)
1. Visit [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Create a free API key
3. Create a file named `.env.local` in the project root
4. Add: `GEMINI_API_KEY=your_actual_api_key`

### Step 2: Start the App (30 seconds)
```bash
npm run dev
```
Open http://localhost:3000 in your browser

### Step 3: Try It Out (1-2 minutes)
1. **Onboarding**: Complete the 6-step profile setup
   - Add your details, interests, and goals
   - Be honest about challenges
   
2. **Dashboard**: View your personalized dashboard

3. **Daily Check-In**: Click "Check In" to log your day

4. **Chat with Mentor**: Click "Chat" and ask anything!
   - "Help me study for my math test"
   - "I'm feeling stressed about exams"
   - "How do I become a software engineer?"

5. **View Report**: Click "Report" to see your progress report
